- 设置用户签名
  ```
  git config --global user.name xxx
  git config --global user.email xxx
  ```
- 初始化本地库
  ```
  git init
  ```
  此时会在工作区文件夹生成隐藏目录.git，暂存区就在这个目录中。
- 查看本地库状态
  ```
  git status
  ```
  会显示不在暂存区但在工作区的文件（红色），在暂存区的文件（绿色）
- git中linux命令通用
  - windows中换行符是CRLF，linux中换行符是LF。
- 将本地文件添加到暂存区
  ```
  git add xxx
  ```
- 删除暂存区文件
  ```
  git rm --cached xxx
  ```
- 将暂存区文件提交到本地库
  ```
  git commit -m "日志信息" xxx
  ```
